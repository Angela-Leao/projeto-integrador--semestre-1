# :houses: GEEAR-Gerenciador de Ongs's
## Resumo

### Este repositório contém o desenvolvimento de um aplicativo para gerenciar Ong's da cidade de Araraquara-SP, com funcionalidades voltadas para o gerenciamento de doações, auxílio na divulgação e mediação de comunicação entre Ong e Doador.
